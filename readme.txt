A simple Elementor widget that adds a Copy to Clipboard button with optional text.

== Description ==

The "Copy to Clipboard Button for Elementor" plugin adds a versatile and user-friendly widget to your Elementor toolkit. This widget allows you to place a customizable button on your pages that, when clicked, copies specified text to the user's clipboard. It's an ideal solution for sharing code snippets, promotional codes, or any text that you want to make easy for visitors to copy.

The widget is highly customizable, offering control over the button's appearance, including text, colors, and styles for both its normal and hover states. You can also display the text to be copied directly on the page, and customize its typography and alignment.

**Key Features:**

*   **Customizable Button:** Change the button text, colors, and styles to match your site's design.
*   **Dynamic Content:** Use Elementor's dynamic tags to populate the text to be copied.
*   **Visible Text Area:** Optionally display the clipboard text on the page.
*   **User Feedback:** The button provides visual feedback to the user upon a successful copy action.
*   **Easy to Use:** Simply drag and drop the widget into your Elementor layout.

== Installation ==

1.  Upload the `copy-to-clipboard-button-for-elementor` folder to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Go to the Elementor editor and find the "Clipboard Button" widget in the widget panel.
4.  Drag the widget to your desired section and start customizing.

== Frequently Asked Questions ==

= Can I use dynamic content in the clipboard text? =

Yes, the clipboard text field supports Elementor's dynamic tags, allowing you to pull in content from various sources.

= How do I change the button's appearance after the text is copied? =

In the widget's "Additional Content" section, you can set the button text and background color that appears after a successful copy.

= Is it possible to hide the text that will be copied? =

Yes, you can toggle the "Show clipboard text" option in the widget's settings to hide or display the text area.
